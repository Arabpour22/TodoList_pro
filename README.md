# Todo List – مدیریت وظایف ساده

یک برنامه کنسولی ساده و کاربردی برای مدیریت لیست کارهای روزانه (To-Do List) نوشته شده با C++.

ویژگی‌های اصلی:
- افزودن وظیفه جدید با عنوان، توضیحات، اولویت و مهلت زمانی
- نمایش همه وظایف
- نمایش وظایف انجام‌شده / انجام‌نشده
- نمایش وظایف عقب‌افتاده (Overdue)
- تغییر وضعیت انجام/انجام‌نشده یک وظیفه
- حذف وظیفه
- هشدار خودکار برای وظایف عقب‌مانده از مهلت

## تکنولوژی‌ها / ابزارها

- C++ (استاندارد مدرن)
- کتابخانه‌های استاندارد: <iostream>, <string>, <vector>, <ctime>
- بدون وابستگی خارجی

## پیش‌نیازها

- کامپایلر C++ که از استاندارد C++11 یا بالاتر پشتیبانی کند  
  (مثال: g++ 7+ ، clang 5+ ، MSVC 2017+)
- سیستم‌عامل: لینوکس / macOS / Windows

## نصب و اجرا

1. فایل پروژه را دانلود یا کپی کنید (مثلاً `main.cpp`)
2. در ترمینال/خط فرمان به پوشه پروژه بروید
3. کامپایل کنید:

   ```bash
   # روش ۱ – ساده
   g++ -o todo main.cpp

   # روش ۲ – با بهینه‌سازی و استاندارد جدید
   g++ -std=c++17 -Wall -O2 -o todo main.cpp
